快速设置子菜单布局选项

用户从主界面选择"快速设置"后，进入子菜单显示预设列表

A

全屏覆盖列表（推荐）

快速设置
预设1: 5V/1A
常规USB供电
预设2: 9V/2A
快充测试
预设3: 12V/3A
LED灯带供电
预设4: 20V/3A
笔记本供电
UP/DOWN选择 | ENTER确认 | EXIT返回

优点

  • 空间充足，可显示详细信息（电压/电流/名称）
  • 视觉焦点集中，不受主界面干扰
  • 符合常见子菜单交互习惯
  • 易于扩展（未来可添加更多预设）

缺点

  • 完全覆盖主界面，无法查看当前输出状态
  • 需要全屏重绘（约86ms）
B

中央弹窗式

LC POWER
[主界面内容被半透明覆盖]
快速设置
1. 5V/1A
2. 9V/2A
3. 12V/3A

优点

  • 保留主界面部分可见性
  • 视觉上更"轻量"，不那么侵入

缺点

  • 实现复杂（需要半透明效果或灰度处理）
  • 240×240屏幕较小，弹窗内空间受限
  • 代码难以维护（需要保存/恢复背景）
  • 刷新开销大（需处理背景+弹窗）

快速设置
5V / 1A

12V / 3A
20V / 3A
UP/DOWN | ENTER | EXIT

优点

  • 视觉极简，字体更大更清晰
  • 代码最简单（无需描述文字）
  • 渲染性能最优

缺点

  • 缺少描述信息，用户需记住预设含义
  • 不够友好（特别是初次使用时）

技术约束

  • 显示硬件: ST7789 240×240 TFT, 软件SPI (~2MHz)
  • 性能: 全屏刷新约86ms, 小文本区域5-10ms
  • 字体: 当前仅支持ASCII 5×7点阵，缩放1x-4x（中文字体未实现）
  • 架构: BSP → Driver → Product层，严格分层，使用Dirty-flag局部刷新
  • 预设数据: 当前硬编码4个预设 (state_machine.c:431-436)
