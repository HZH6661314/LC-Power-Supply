## 架构分层

```
应用层 (Application)
  └── state_machine.c     (状态管理、UI 状态)
产品层 (Product)
  ├── ui_display.c        (UI 渲染逻辑)
  └── key.c              (按键输入处理)
控制层 (Control)
  └── power_control.c    (恒压/恒流控制)
驱动层 (Driver)
  ├── tft_driver.c       (TFT 硬件接口)
  └── tft_gfx.c          (图形原语/字体)
BSP 层
  ├── bsp_lcd.c          (LCD 硬件抽象)
  ├── bsp_adc.c          (电压/电流采样)
  ├── bsp_hrtim.c        (PWM 控制 + 系统 Tick)
  └── bsp_gpio.c         (GPIO 抽象)
调度层 (Scheduler)
  └── task_manager.c     (任务调度：1ms/10ms/50ms/100ms)
```
