# LC电源项目计划

## 项目概览
**项目名称**: 100W数控电源
**MCU**: STM32F334
**显示**: 240×240 TFT LCD (ST7789)
**SPI类型**: 软件SPI（位拍）
**当前阶段**: UI闪烁消除 + 光标闪烁修复 (V10)

## 架构分层

```
应用层 (Application)
  └── state_machine.c     (状态管理、UI 状态)
产品层 (Product)
  ├── ui_display.c        (UI 渲染逻辑)
  └── key.c              (按键输入处理)
控制层 (Control)
  └── power_control.c    (恒压/恒流控制)
驱动层 (Driver)
  ├── tft_driver.c       (TFT 硬件接口)
  └── tft_gfx.c          (图形原语/字体)
BSP 层
  ├── bsp_lcd.c          (LCD 硬件抽象)
  ├── bsp_adc.c          (电压/电流采样)
  ├── bsp_hrtim.c        (PWM 控制 + 系统 Tick)
  └── bsp_gpio.c         (GPIO 抽象)
调度层 (Scheduler)
  └── task_manager.c     (任务调度：1ms/10ms/50ms/100ms)
```

## 任务调度配置

- 状态机: 1ms（控制环路关键）

- UI 刷新: 50ms（20Hz 更新速率）
