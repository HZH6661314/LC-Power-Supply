# 任务 2：重构 UI 交互状态机（核心）
必须废弃现有的面条式代码，使用严谨的状态机（State Machine）重构按键逻辑。系统共有 4 个物理按键：`SET`, `EXIT`, `UP`, `DOWN`。

## 状态 0：主界面-待机/运行态 (UI_STATE_HOME_IDLE)
- **视觉**：正常显示电压电流等，**没有任何光标**。
- **SET 按键**：短按切换电源输出状态（待机关闭 <-> 软启动开启输出）。
- **UP/DOWN 按键**：短按唤醒光标，状态跳转至 `UI_STATE_HOME_MENU`，默认光标落在“设置电压”上。
- **EXIT 按键**：无动作。

## 状态 1：主界面-菜单选择态 (UI_STATE_HOME_MENU)
- **视觉**：当前选中项的矩形框**反相显示**（光标）。
- **UP/DOWN 按键**：短按在 4 个选项中循环移动光标（1.设置电压 -> 2.设置电流 -> 3.快速设置 -> 4.设置 -> 回到1）。
- **SET 按键**：
  - 若光标在 1 或 2，状态跳转至 `UI_STATE_HOME_EDIT`。
  - 若光标在 3，状态跳转至 `UI_STATE_QUICK_SET`。
  - 若光标在 4，状态跳转至 `UI_STATE_SYS_SET`。
- **EXIT 按键**：短按光标消失，状态退回 `UI_STATE_HOME_IDLE`。若参数有修改，触发 W25Q256 异步保存（断电不丢失）。

## 状态 2：主界面-参数编辑态 (UI_STATE_HOME_EDIT)
- **视觉**：当前选中项的光标以 **2Hz 频率闪烁**。
- **UP/DOWN 按键**：短按/长按用于增减电压或电流的数值。
- **SET 按键**：确认当前修改，光标停止闪烁，状态退回 `UI_STATE_HOME_MENU`。
- **EXIT 按键**：
  - **长按**：取消本次修改（恢复修改前的值），直接退回 `UI_STATE_HOME_IDLE`。

## 状态 3：快速设置子菜单 (UI_STATE_QUICK_SET)
- **视觉**：清屏并显示 4 个快速设置参数组（读取自 W25Q256 的预设 V/I 值）。进入时**无光标**。
- **UP/DOWN 按键**：唤醒并上下移动光标（反相显示）。
- **SET 按键**：选中当前组，UI 渲染一个“箭头(->)”指向该组，代表应用此组参数到主输出。
- **EXIT 按键**：
  - **短按**：若有光标则隐藏光标；若无光标则退回 `UI_STATE_HOME_MENU`。
  - **长按**：无视光标状态，直接退回主界面的 `UI_STATE_HOME_IDLE`。

## 状态 4：系统设置子菜单 (UI_STATE_SYS_SET)
- **视觉**：列表式显示 8 个子菜单项：1.快设参数编辑, 2.手动校准, 3.过流保护(OCP), 4.过压保护(OVP), 5.过功率保护(OPP), 6.过热保护(OTP), 7.版本信息, 8.退出。
- **交互逻辑**：
  - **UP/DOWN 按键**：在 8 个选项中上下移动光标。
  - **SET 按键**：
    - 若选 1~6：进入二级数值调节态（光标 2Hz 闪烁，UP/DOWN 调值，SET 确认，与状态 2 逻辑一致）。
    - 若选 7：仅作静态信息展示。
    - 若选 8：退回 `UI_STATE_HOME_MENU`。
  - **EXIT 按键**：无论在何处，长按直接退回主界面 `UI_STATE_HOME_IDLE`；短按退回上一级。
